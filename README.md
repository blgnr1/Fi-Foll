# Fi-Foll 💳
Fi-Foll, özellikle sosyal etkinliklerde ve grup gezilerinde ortaya çıkan karmaşık harcama takibi sorununu çözmek için tasarlanmış, minimalist bir finansal takip uygulamasıdır. Sade ve basit bir kullanıcı deneyimiyle, borçlarınızı ve bakiye durumunuzu anlık olarak yönetmenizi sağlar.

✨ Öne Çıkan Özellikler
-

Harcama ve Borç Takibi: Arkadaşlarınızla yaptığınız gezilerde veya günlük hayatta kimin ne kadar harcadığını ve kimin kime borçlu olduğunu kolayca takip edin.

Hesap Özeti: Finansal durumunuzu gösteren temiz bir hesap özeti.

Döviz Entegrasyonu: Güncel kurlar üzerinden harcama takibi yapabilmek için API desteği.

🛠 Teknik Yığın (Tech Stack)
-
Framework: Flutter & Dart

State Management: Riverpod

Local Storage: SQLite

🚀 Kurulum
-
Projeyi yerel makinenizde çalıştırmak için:

Repoyu klonlayın:   `git clone https://github.com/blgnr1/fi-foll.git`

Bağımlılıkları yükleyin:    `flutter pub get`

Uygulamayı çalıştırın:    `flutter run`
